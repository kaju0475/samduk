# 삼덕용기 시스템 이전(Migration) 가이드

현재 사용 중인 시스템(`samduk-system`)을 다른 컴퓨터로 안전하게 옮기는 방법입니다.

## 1. 준비 사항 (새 컴퓨터)

새 컴퓨터에 **Node.js**가 설치되어 있어야 합니다.

- **다운로드**: [Node.js 공식 홈페이지](https://nodejs.org/) (LTS 버전 권장)
- 설치 후 터미널(CMD)에서 `node -v`를 입력하여 버전이 나오는지 확인하세요.

## 2. 파일 복사 및 백업

가장 중요한 것은 데이터 파일인 `db.json`입니다.

1.  **전체 폴더 압축**:
    - 현재 컴퓨터의 `samduk-system` 폴더 전체를 압축합니다. (`samduk-system.zip`)
    - _주의_: `node_modules` 폴더와 `.next` 폴더는 용량이 매우 크므로 제외하고 압축하는 것이 이동에 유리합니다. (새 컴퓨터에서 다시 설치하면 됩니다.)
    - **필수 포함 파일**:
      - `db.json` (데이터베이스)
      - `package.json` (설정)
      - `app/`, `public/`, `components/`, `lib/` 등 소스 코드 폴더

2.  **데이터 별도 백업 (권장)**:
    - 만약을 대비해 `db.json` 파일은 USB나 클라우드에 따로 한 번 더 복사해 두시기 바랍니다.

## 3. 새 컴퓨터로 이동 및 설치

1.  새 컴퓨터의 원하는 위치(예: 바탕화면)에 압축을 풉니다.
2.  폴더 안으로 들어갑니다 (예: `C:\Users\User\Desktop\samduk-system`).
3.  폴더의 빈 곳을 `Shift + 우클릭` -> **"PowerShell 창 열기"** 또는 **"터미널 열기"**를 선택합니다.
4.  아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
    ```bash
    npm install
    ```
    (인터넷 연결이 필요하며, 몇 분 정도 소요됩니다.)

## 4. 시스템 실행

설치가 완료되면 아래 명령어로 시스템을 켭니다.

```bash
npm run dev
```

- **접속 주소**: 인터넷 브라우저(Chrome/Edge)를 켜고 `http://localhost:3000` 으로 접속합니다.
- **모바일 접속**: 같은 와이파이를 쓰고 있다면 `npm run dev` 실행 시 표시되는 `Network` 주소(예: `http://192.168.0.x:3000`)로 핸드폰에서 접속할 수 있습니다.

## 5. (선택사항) 데이터 복구

만약 이동 후 데이터가 보이지 않는다면, 백업해 둔 `db.json` 파일을 새 컴퓨터의 프로젝트 폴더 덮어쓰기 하시면 됩니다.
프로젝트 루트 경로: `.../samduk-system/db.json`

---

> **팁**: 옮긴 후에도 `scripts/backup_db.ts` 등을 활용해 주기적으로 백업을 수행하는 것이 좋습니다.
